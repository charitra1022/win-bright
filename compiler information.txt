to compile gtk app:

1. Install msys4
2. install gtk4, gtkmm4, mingw-toolchain using msys4
3. set mingw compiler as default compiler in codeblocks
4. create a Console Application project with c++ in codeblocks.
5. Open <Project> > Build Options and follow the listed.

6. Compiler Settings > Other Comipler Options

`pkgconf --cflags gtk4`
`pkgconf --cflags glib-2.0`
`pkgconf --cflags glibmm-2.68`
`pkgconf --cflags gtkmm-4.0`
`pkgconf --cflags gdkmm-2.4`

7. Linker Settings > Other Linker Options

`pkgconf --libs gtk4`
`pkgconf --libs glib-2.0`
`pkgconf --libs glibmm-2.68`
`pkgconf --libs gtkmm-4.0`
`pkgconf --libs gdkmm-2.4`
